## Introduction

This project is a web calculator built with HTML, CSS, and JavaScript. It provides a user-friendly interface for basic arithmetic operations.

## Features

- Addition, subtraction, multiplication, and division operations.
- Clear display and responsive layout.
- Designed for desktop and tablet devices.

## Getting Started

To get a copy of this project and run it on your local machine, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/web-calculator.git
   ```

2. Open the project folder:

   ```bash
   cd web-calculator
   ```

3. Open the `index.html` file in your web browser.

## Usage

- Enter numbers using the numeric buttons.
- Use the operator buttons for addition (+), subtraction (-), multiplication (*), and division (/).
- Press the "=" button to calculate the result.
- Press "C" to clear the display.
